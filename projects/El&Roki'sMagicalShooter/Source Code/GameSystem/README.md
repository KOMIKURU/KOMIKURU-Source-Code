# GameSystem

ゲームの流れや演出、スコアを管理するシステム。

## 📁 ファイル構成

```
GameSystem/
├── BGMManager.cs
├── BackgroundScroll.cs
├── ESManager.cs
├── GameFlowManager.cs
├── ImageBlinker.cs
├── SEManager.cs
├── TMPBlinker.cs
└── UISlideIn.cs
```

### コードの説明
| ファイル | 説明 |
|---------|------|
| BGMManager.cs | BGMを管理 |
| BackgroundScroll.cs | 背景のスクロールを管理 |
| ESManager.cs | 環境音を管理 |
| GameFlowManager.cs | ゲーム全体の流れを管理 |
| ImageBlinker.cs | イメージを点滅させる |
| SEManager.cs | SEを管理 |
| TMPBlinker.cs | TMProを点滅させる |
| UISlideIn.cs | 画像やパネルをスライドインさせる |
