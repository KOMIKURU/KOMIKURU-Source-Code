## 💡 作品の工夫点

### 1. 操作端末にJoyconを利用
ニンテンドースイッチのコントローラーとして知られるJoyConの「誰もが見たことある」という親近感を逆手に取ったアイデアであり、実際ジョイコンなら操作できるからとプレイしてくれた人も多かった。

### 2. シンプルなキャラクターデザイン
他の展示もある中、大多数の目を引くには「一目でそれが何か分かるもの」あるいは「ぐちゃぐちゃとしていて何を見たらいいのかわからないデザインを避けること」が重要であると私は考えた。結果的に、子供たちへの受けがよく、彼らに同伴していた保護者と一緒に遊ぶという光景はよく見られた。

### 3. ゲームデザイン
世の中にはゲームが得意な人と苦手な人が存在する。そして、いばりつには幅広い年齢層の人々が訪れる。複雑で長いゲームプレイはゲームが苦手な人には苦痛である。待ち時間が長かった場合、心理的な負担はさらに高まるだろう。そこで私は思い切ってシンプルなボタン操作かつ40秒1プレイという超短いゲームデザインを採用した。これは大衆を呼び込むデザインとして非常に優れており、結果として1日で341人に遊んでもらえた。
