# KOMIKURU Source Code Repository

ここは、中野陽（KOMIKURU）が開発したゲーム作品の
**ソースコード**を公開するリポジトリです。

## 📚 収録プロジェクト

### Pandora
「行こう、心を求める旅に。」
神託を胸に、エルフの少女エルと共に荒廃した神々の王国"オリンパス"へと旅立とう。
数多の試練と異形なる住人達との対話の果てにオリンパスの真実を暴き出せ。

## 🎯 このリポジトリの目的

情報系の学生として、以下のようなスキルを
評価していただくために公開しています：

- **システム設計**: ステートマシン、ゲーム管理など
- **プログラミングスキル**: C#、Unity での実装方法
- **課題解決能力**: 複雑な仕様実現のための工夫
- **コード品質**: 保守性、拡張性を考慮した設計

## ⚖️ 法的注記

各プロジェクトの README に詳しく記載していますが、
以下をご理解ください：

- Pandora は Unity 有料アセット使用のため、完全なゲーム構成ではなく
  **コードのみ**の公開です
