## 🎯 このリポジトリの目的

情報系の学生として、以下のようなスキルを
評価していただくために公開しています：

- **システム設計**: ステートマシン、ゲーム管理など
- **プログラミングスキル**: C#、Unity での実装方法
- **課題解決能力**: 複雑な仕様実現のための工夫
- **コード品質**: 保守性、拡張性を考慮した設計

## ⚖️ 法的注記

各プロジェクトの README に詳しく記載していますが、
以下をご理解ください：

- Pandora は Unity 有料アセット使用のため、完全なゲーム構成ではなく
  **コードのみ**の公開です
- エルとロキ以降は、著作権上問題がない範囲で
  **完全なソースコード + ゲーム実行ファイル**を公開予定です
